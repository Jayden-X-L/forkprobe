三类高频任务

网络上的skill百花齐放，使用各种skill 之前，先知道哪一个真的帮得上忙。

forkprobe 不是再造一个写作助手，而是把多个候选 skill 变成可观察、可比较、可选择的工作流。

办公方向

产品方案、GTM、会议纪要、市场分析和汇报 PPT，不再反复试提示词，而是直接比较几条 skill 路线。

科研方向

中文 SCI 润色、英文摘要、Nature 风格、审稿回复、figure storyline 和科研 PPT，比较表达效果同时守住技术含义。

金融方向

持仓数据、行业报告、财报摘要、风险敞口和投委会材料，让多个分析路径并行跑，再选证据链最稳的一版。

实际使用时

使用 forkprobe，同一个任务、多个 skill 的结果清晰可见，轻松选择 winner。

左侧是原始输入和候选 skill，中间展示每一路完整输出，右侧给出 AI 评审建议。选定 winner 后，forkprobe 会生成 continuation handoff，让当前会话沿着你选中的结果继续执行。
