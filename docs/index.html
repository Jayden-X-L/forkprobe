三类高频任务

网络上的 skill 百花齐放，使用各种 skill 之前，先知道哪一个真的帮得上忙。

forkprobe 不是再造一个写作助手，而是先读取任务信号、推荐候选 skill，再把这些候选变成可观察、可比较、可选择的工作流。

办公方向

产品方案、GTM、会议纪要、市场分析、调研报告和汇报 PPT，不再反复试提示词，而是直接比较几条 skill 路线。

产品方案 内部宣传 PPT

科研方向

中文 SCI 润色、英文摘要、Nature 风格、审稿回复、论文作图 / 科研绘图和科研 PPT，比较表达效果与成品质量，同时守住技术含义。

SCI 写作 科研绘图 组会材料

金融方向

持仓数据、行业报告、财报摘要、投研报告、风险敞口和投委会材料，让多个分析路径并行跑，再选证据链最稳的一版。
